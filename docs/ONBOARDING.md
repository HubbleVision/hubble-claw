# Hubble Claw 用户接入说明

> 知秋 (Vane) 🍂 — 基于 OpenClaw 的股票投研分析师 Agent

---

## 1. 快速开始

### 1.1 环境准备

1. 部署 OpenClaw Agent 平台
2. 复制 `.env.example` 为 `.env`，填入必要配置：

```env
MARKET_API_BASE_URL=<Hubble 数据服务地址>
MARKET_API_KEY=<你的 API Key>
TAVILY_API_KEY=<Tavily 搜索 API Key>
```

3. 选择模型：默认 MiniMax M2.7
4. 连接微信渠道（扫码绑定）

### 1.2 首次对话

Agent 首次启动会读取 `BOOTSTRAP.md` 进行初始化对话，完成后自动删除。初始化内容包括：
- 确认 Agent 身份（已预设为"知秋"）
- 了解用户背景（投资风格、关注市场）
- 生成 `USER.md` 记录用户画像

### 1.3 Skill 安装

所有核心技能已预置安装（见 `skills_list.md`）。如需安装新技能：
- 在微信聊天中直接发送 skill 链接或名称
- Agent 会自动调用 `skill-vetter` 进行安全审查
- 审查通过后确认安装

## 4. 进阶用法

### 4.1 多技能协作

复杂问题会自动触发多技能协作。例如：

**"最近半导体行业有什么大事，帮我做个分析"** 会按以下顺序执行：
1. `alphaear-news` → 新闻聚合 + 趋势汇总
2. `market-stock` → 相关个股数据 + 技术指标
3. `alphaear-reporter` → 信号聚类 → 研报生成

### 4.2 记忆系统

- **短期记忆**：每日对话自动记录在 `memory/YYYY-MM-DD.md`
- **长期记忆**：关键偏好和历史分析沉淀在 `MEMORY.md`
- **持仓跟踪**：通过 `watchlist.md` 跟踪关注的标的

你可以说 **"记住我偏好价值投资"** 或 **"把宁德时代加到我的关注列表"**，Agent 会自动更新相应文件。

### 4.3 数据来源

所有行情数据来自 **Hubble 私有数据服务**，实时更新，覆盖 A股、港股、美股三大市场。

---

## 5. 注意事项

1. **Agent 不会给出直接买卖指令** — 提供分析和数据，决策由用户做出
2. **数据时效性** — Agent 会标注数据截止时间，过时数据会主动提示
3. **市场覆盖** — 当前支持 A股、港股、美股，暂不支持加密货币和期货衍生品
4. **隐私保护** — 群聊中不会透露用户持仓、盈亏等私密信息
5. **分析置信度** — 每次分析会标注 "高置信"/"中等"/"投机性" 等级
