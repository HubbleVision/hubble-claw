# 投研龙虾 Demo 演示场景

> 按能力维度设计，从单一 skill 到多 skill 编排，逐步展示系统能力。

---

## 场景一：个股快速诊断（单 skill 入门）

**演示能力：** `market_stock` 基础行情 + 技术指标

**提问：**
```
茅台最近一个月走势怎么样？帮我看下 MACD、RSI 和布林带
```

**看点：** 并行拉取 K 线 + 批量指标，一次性输出图表级分析

---

## 场景二：财报季复盘（多 skill 并行）

**演示能力：** `earnings-recap` + `estimate-analysis` + `market_stock`

**提问：**
```
英伟达最近三个季度财报是 beat 还是 miss？分析师对下季度的 EPS 预期是多少？股价在财报后分别怎么反应的？
```

**看点：** 三个 skill 并行取数 → 串行综合分析，展示编排能力

---

## 场景三：行业横向对比（多股并行 + 流动性）

**演示能力：** `market_stock` × N 并行 + `stock-liquidity`

**提问：**
```
帮我对比一下宁德时代、比亚迪、理想汽车这三家的近三个月走势和流动性，谁更适合短线交易？
```

**看点：** 3 只股票数据并行拉取 + 6 维流动性分析 → 对比表格输出

---

## 场景四：热点事件驱动分析（新闻 + 行情联动）

**演示能力：** `alphaear-news` + `market_stock` + `tavily`

**提问：**
```
今天 A 股半导体板块有什么重大消息？相关龙头股表现如何？
```

**看点：** 新闻聚合 → 自动识别相关个股 → 拉取实时行情，展示事件驱动分析链路

---

## 场景五：深度研报生成（全链路编排 - 王炸场景）

**演示能力：** `alphaear-news` ∥ `market_stock` ∥ `estimate-analysis` ∥ `earnings-recap` → `alphaear-reporter`

**提问：**
```
帮我生成一份腾讯控股的投研报告，包括近期走势、技术面、基本面、市场情绪和分析师观点
```

**看点：** 5 个 skill 协同，信号聚类 → 多维分析 → 报告组装，展示系统最强编排能力

---

## 场景六：盘前 / 盘后大盘总览（心跳场景）

**演示能力：** `market_stock`（全指数并行）+ `alphaear-news`

**提问：**
```
现在各大市场情况怎么样？A 股、港股、美股主要指数涨跌如何？有什么值得关注的？
```

**看点：** 三个市场指数并行拉取 + 新闻 → 一页式市场综述

---

## 场景七：微信研报解读（内容获取 + 分析）

**演示能力：** `wechat-article-extractor` + `market_stock`

**提问：**
```
帮我读一下这篇公众号文章 https://mp.weixin.qq.com/s/xxxxx ，里面提到的个股帮我拉一下最新数据看看靠不靠谱
```

**看点：** 自动识别微信链接 → 提取文章 → 识别提及个股 → 拉取实时数据交叉验证
